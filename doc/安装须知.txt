如果有server或database编码不是utf8，使用以下命令更改
SET character_set_server = utf8 ;
SET character_set_database = utf8 ;

有的数据库无法直接使用命令更改成功，请更改my.ini配置文件的编码
[mysqld]
character-set-server=utf8

[client]
default-character-set=utf8

[mysql]
default-character-set=utf8
